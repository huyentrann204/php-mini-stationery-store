# PHP Mini Stationery Store App
Week 1 PHP Lab
## Các tính năng:
- **Dashboard:** Giao diện trang chủ hiện đại
- **Quản lý sản phẩm:** Danh sách sản phẩm với các thông tin chi tiết và phân loại trạng thái.
- **Báo cáo và phân tích:** Trang Reports riêng biệt sử dụng Chart.js hiển thị biểu đồ tỷ lệ hàng hóa trực quan.

## Run 
Chạy lệnh: `php -S localhost:8000 -t public`
Truy cập trình duyệt tại địa chỉ: `http://localhost:8000`
